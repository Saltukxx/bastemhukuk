# Bastem Hukuk Bürosu

Saf HTML, CSS ve JavaScript ile hazırlanmış kurumsal hukuk bürosu sitesi.

## Çalıştırma

`index.html` dosyasını tarayıcıda açın. İsteğe bağlı:

```bash
python3 -m http.server 8000
```

## Sayfalar

- `index.html` — Anasayfa
- `hakkimizda.html` — Hakkımızda
- `calisma-alanlari.html` — Çalışma alanları
- `blog.html` / `blog-detay.html` — Yazılar
- `iletisim.html` — İletişim
- `randevu.html` — Görüşme talebi

Formlar istemci tarafında doğrulanır; backend yoktur.
